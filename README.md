icecube/icetray
===============

Docker images of IceCube's IceTray codebase - combo metaproject.

Flavors
-------

slim: just icetray

prod: slim + tables and GCD files

devel: prod + compiler toolchain and test data

cudaX: devel + cuda support (X = cuda version)
